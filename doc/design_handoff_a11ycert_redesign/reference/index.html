실전형 모의 퀴즈

실제 시험과 동일한 4지선다 형식. 즉시 채점과 영역별 정답률 분석으로 약점을 파악합니다.

→ 즉시 채점 + 해설
→ 자동 오답 노트

간격 반복 플래시카드

기억 곡선에 맞춰 카드가 다시 등장합니다. 하루 10분으로 핵심 200+ 용어를 장기 기억에 정착시키세요.

→ 다음 복습일 자동 계산
→ 즐겨찾기 + 형광펜

합격까지의 4단계

평균 6주, 하루 30분 학습 기준

  1. 1 기본 개념 학습 자격증 개요와 핵심 영역을 이중 언어 카드로 훑습니다.
  2. 2 용어 암기 플래시카드로 출제 빈도가 높은 용어를 반복합니다.
  3. 3 영역별 퀴즈 영역마다 25문항 단위로 풀고 약점을 보강합니다.
  4. 4 실전 모의시험 실제 시험과 동일한 시간·환경으로 마무리 점검.

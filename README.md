# A11yCert

IAAP CPACC & WAS 자격증 시험 준비를 위한 한국어·영어 이중 언어 학습 플랫폼.

**[a11ycert.com](https://a11ycert.com)** — Korean / English bilingual study platform for IAAP CPACC & WAS certification exams.

---

## Features

- 한국어·영어 이중 언어 학습 카드
- 플래시카드 (핵심 용어 암기)
- 모의 퀴즈 (CPACC 100문항 / WAS 75문항)
- 한영 용어집 (500개 이상)
- 학습 진도 저장 (localStorage 기반, 로그인 불필요)

---

## Tech Stack

- **Next.js 16** — App Router, Static Export
- **next-intl** — `/ko` / `/en` i18n routing
- **Tailwind CSS v4**
- **Zustand** — client-side state
- **GitHub Pages** — free hosting via GitHub Actions
